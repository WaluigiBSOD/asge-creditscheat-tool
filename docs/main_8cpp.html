◆ _InitializeAllThatNeedsToBeInitialized()

bool _InitializeAllThatNeedsToBeInitialized ( )

Allocates all the arrays that are put inside the heap.

It's done to avoid memory allocation issues, as in total they weight 64 KB. May sound exaggerate, but usually default stack sizes are limited to a few megabytes. Also, this is approximately 420 times the RAM that is needed for the internal state of a MT19937 PRNG algorithm instance, as a comparison.

Returns
True if all memory needed is allocated and initialized succesfully, false otherwise.
